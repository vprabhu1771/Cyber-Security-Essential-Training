# 1 - Integer Input and Output

```bash
echo "Enter x value"

read x

echo "Enter y value"

read y

echo "x = $x"

echo "y = $y"
```

OUTPUT

```
Enter x value
10

Enter y value
5

x = 10

y = 5
```

# 2 - String Input and Output

```bash
echo "Enter your name"

read name

echo "your name is $name"
```

OUTPUT

```
Enter your name
prabhu

your name is prabhu
```
